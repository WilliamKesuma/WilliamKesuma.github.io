# 💻 William Kesuma - Personal Portfolio v1.0

This repository contains the source code for my professional portfolio website. The site is designed to showcase my expertise as a Full-stack Developer and aspiring Business Analyst, featuring my work from the Apple Developer Academy and my research in Data Science.

**🌐 Live Demo:** [https://WilliamKesuma.github.io](https://WilliamKesuma.github.io)

---

## 🚀 Overview
The goal of this project was to create a minimalist, high-performance landing page that reflects my design philosophy: **clean, functional, and data-driven.** It serves as a central hub for my technical projects, AWS certifications, and academic research.

## 🛠 Tech Stack
- **Architecture:** HTML5, CSS3, JavaScript (Vanilla)
- **Deployment:** GitHub Pages
- **Design:** Modern Minimalist (Responsive Design)
- **Version Control:** Git (managed with professional commit history)
